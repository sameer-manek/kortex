# kortex

a PHP based MVVM framework that has built in:

 *encryption
 *token management
 *input sanitization
 *DB Engine
 *Cookie management
 
*and many more features built in for security and backend work.*

for generating the view, you just create the pages (index.php, home.php, etc) in the root directory. and include the core/init.php [configuration file] in the beginning of every page. you can create as many classes in the classes dir as you want and it shall work.
